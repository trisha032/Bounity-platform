🏆 Bounty Platform (Soroban Smart Contract)

📌 Project Description

The Bounty Platform is a decentralized application built on the Stellar Soroban smart contract platform. It allows users to create, manage, and complete task-based bounties in a secure and trustless environment without intermediaries.

🚀 What It Does

- Users can create bounties with rewards
- Participants can complete tasks
- Tracks bounty completion on-chain
- Stores data securely using smart contracts

✨ Features

- 🧾 Create Bounty
- 👨‍💻 Complete Bounty
- 🔍 On-chain transparency
- 🔐 Secure authentication using Stellar addresses
- 📦 Decentralized storage
